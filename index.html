A declaração informa ao navegador que o documento é um documento HTML5.

O elemento é o elemento raiz do documento HTML.

O elemento contém informações sobre o documento, como o título e a folha de estilo.

O elemento especifica o título do documento.</p> <p>O elemento <meta> especifica metadados sobre o documento, como a largura da viewport e a escala inicial.</p> <p>O elemento <link> especifica um link para uma folha de estilo.</p> <p>O elemento <body> contém o conteúdo principal do documento.</p> <p>O elemento <header> contém o cabeçalho do documento, que geralmente inclui o título e os links de navegação.</p> <p>O elemento <h1> especifica um cabeçalho de nível 1.</p> <p>O elemento <nav> contém os links de navegação do documento.</p> <p>O elemento <a> especifica um hiperlink.</p> <p>O elemento <main> contém o conteúdo principal do documento.</p> <p>O elemento <h2> especifica um cabeçalho de nível 2.</p> <p>O elemento <p> especifica um parágrafo.</p> <p>O elemento <footer> contém o rodapé do documento, que geralmente inclui informações de direitos autorais.</p> </main> <footer> <p>Copyright © 2023 Adrianinho</p> </footer> </body> </html>
